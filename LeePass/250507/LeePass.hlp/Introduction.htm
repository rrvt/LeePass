LeePass

Introduction

KeePass is an open source password vault. It is composed of two parts: a dynamic link library (dll) that encrypts and decrypts the data and a user interface. LeeApp is just a different user interface to the KeePass dll.

The app shows one record in the window. There are two drop down lists on the tool bar that govern which entries viewed in the window. The first drop down list lists the groups one of which is "All Groups". The second drop down list is the "Entries" list. It lists the entries in the group showing in the Group drop down list edit box on the tool bar. Once a grou is chosen then an entry may be displayed by:

  • Picking an entry from the toolbar Entries drop down list
  • Search for a specific entry with the search command (). The search may be performed according to several constraints or attributes specified in the search dialog box. There may be more than one record that match the seach string. The "Search Next" command () will find the next match.
  • Browse through the entries in the Entries drop down list by using the "<" (e.g. "Previous") and ">" ("Next") toolbard commands. The initial value is the first item on the Entry drop down list.

The operations available are discussed on the operations page.
