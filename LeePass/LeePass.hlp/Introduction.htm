LeePass

Introduction

KeePass is an open source password vault. It is composed of two parts: a dynamic link library (dll) that encrypts and decrypts the data and a user interface. LeeApp is just a different user interface to the KeePass dll.

The app shows one record in the window. There are two drop down lists on the tool bar that govern which entries viewed in the window. The first drop down list lists the groups one of which is "All Groups". The second drop down list is the "Entries" list. It lists the entries in the group showing in the Group drop down list edit box on the tool bar. Once a grou is chosen then an entry may be displayed by:

  • Picking an entry from the toolbar Entries drop down list
  • Search for a specific entry with the search command (). The search may be performed according to several constraints or attributes specified in the search dialog box. There may be more than one record that match the seach string. The "Search Next" command () will find the next match.
  • Browse through the entries in the Entries drop down list by using the "<" (e.g. "Previous") and ">" ("Next") toolbard commands. The initial value is the first item on the Entry drop down list.

When an entry is visible in the app three commands on the menu bar will help view the web site and login. The commands are:

  • Start URL -- Send url to the default browser
  • Copy UserName -- Copies the User Name to the clipboard
  • Copy Password -- Copies the password to the clipboard for 30 seconds

When the app starts it is in Read Only mode. One can browse or find entries make changes in the entries and when the app exits nothing will be saved. To edit an entry or create a new one one must execute the toggle Save/Read Only State command (). One feature of the Toggle command is that when the app is in the Save State a button () will appear next to the password. The key button, when pressed will generate a new password and insert it into the entry's password box. A new password may be generated using the menu command Tools/Generate Password.

House Keeping

As with all databases there are a slew of housekeeping commands. One can create a new database, open an existing database, login in to the currently selected database, create a new entry, create a new password for an existing entry, explicitly save an entry, delete an entry, delete a group of entries (i.e. delete a group).

The status line on the bottom border of the app shows the current state of the app as best as can be executed. The path to the database, the number of groups and entries in the database is displayed in the right half. The current record and Save/Read Only state is shown in the left half. Copying entities to the clipboard is shown in the left half.

LastPass

LastPass is a commercial password vault and integrates with Firefox nicely. This vault is a backup. Export the entities from LastPass and import them into LeePass. One can also export from LeePass and import into LastPass. The resulting entries in LastPass don't appear to be exactly the same as the entries that were exported from LastPass but they are similar. The commands are on the toolbar in a drop down menu captioned LastPass:

  • Import Lastpass File -- reads the Comma Separated Value (csv) file
  • Export a Lastpass File -- writes a Comma Separated Value (csv) file
  • Expunge File -- writes random characters over the content of the file, then deletes it

The operations available are discussed on the operations page.
